#### 节点类型

- 节点类型
    - ZHI_QI = 1; // 智启
    - ZHI_HUI = 2; // 智汇
    - ZHI_CE = 3; // 智策
    - ZHI_DING = 4; // 智鼎

#### 购买节点

- 方法
    - function mint(
      uint256 uid,
      address user,
      uint256 nodeId,
      uint256 amount
      ){}
- 参数
    - uid 用户UID
    - user 用户钱包地址
    - nodeId 节点ID
    - amount 购买数量

#### 批量购买节点

- 方法
    - function mintBatch(
      uint256[] calldata uids,
      address[] calldata users,
      uint256[] calldata nodeIds,
      uint256[] calldata amounts
      )
- 参数，参数同一个索引是同一个用户
    - uids 用户UID数组
    - users 用户钱包地址数组
    - nodeIds 节点ID数组
    - amounts 购买数量数组

#### 销毁已购买节点

- 方法
    - function burnFrom(
      uint256 uid,
      address user,
      uint256 nodeId,
      uint256 value
      ){}
- 参数
    - uid 用户UID
    - user 用户钱包地址
    - nodeId 节点ID
    - amount 购买数量
- 备注：如果购买错了，可以调用此方法，清除购买

#### 用户查询已购节点

- 方法
    - function userBalance(
      uint256 uid,
      uint256 nodeId
      ) public view returns (uint256){}
- 参数
    - uid 用户UID
    - nodeId 节点ID
- 返回值
    - 用户该类型节点购买数量

#### 设置token URI

- 方法
    - function setURI(
      string calldata newuri
      ){}
- 参数
    - newuri token的URI信息
